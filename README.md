# 📊 Regional Sales Performance Dashboard

## 🚀 Data Analyst Portfolio Project

---

## 📌 Project Overview
This project is an interactive **Regional Sales Performance Dashboard** that analyzes year-to-date sales data across 5 regions. It helps identify top-performing areas, detect declining trends, and generate actionable business insights.

📅 **Data Period:** Jan 2024 – Dec 2024  
📊 **Regions:** 5  
📁 **Dataset Size:** 60+ records  

---

## 🎯 Business Problem
Organizations need clear visibility into regional sales performance to:
- Identify top and underperforming regions  
- Understand growth trends  
- Optimize resource allocation  
- Improve decision-making  

---

## 📈 Key Insights
- 💰 **Total Sales:** $7.17M  
- 📊 **Growth:** +12.4% YoY  
- 🥇 **Top Region:** West ($1.57M, +4.1%)  
- ⚠️ **At Risk:** North (-0.5%)  

---

## 📊 Dashboard Features

- 📌 KPI Cards (Sales, Growth, Performance)
- 📊 Bar Chart → Regional sales comparison  
- 📈 Line Chart → Monthly sales trends  
- 🔵 Bubble Chart → Growth vs Sales analysis  
- 📋 Performance Table with ranking  
- 🎯 Actionable insights section  

---

## 🛠️ Tech Stack

- **Data Processing:** Python (Pandas)  
- **Visualization:** Chart.js  
- **Frontend:** HTML, CSS, JavaScript  
- **Data Source:** CSV file

## 📌 Key Findings

- West region is consistently the top performer  
- Central region has the highest order value  
- East and North show declining trends
